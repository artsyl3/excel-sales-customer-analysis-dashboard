# excel-sales-customer-analysis-dashboard
End-to-end Excel data analysis project including data cleaning, transformation, pivot tables, and an interactive dashboard to uncover customer purchasing insights.

📊 Excel Data Analysis & Dashboard Project
🔍 Overview

This project showcases a complete end-to-end data analysis workflow in Excel, starting from raw data and ending with an interactive dashboard.

The goal of this project is to analyze customer demographics and identify patterns that influence bike purchases.

🧰 Tools & Skills Used

Microsoft Excel

Data Cleaning

Data Transformation

Pivot Tables

Dashboard Design

Data Visualization

IF Functions & Logical Formulas

📁 Dataset

The dataset contains customer information such as:

Age

Income

Marital Status

Education

Occupation

Region

Purchase Behavior (Bike Buyer)

⚙️ Process
1. Data Cleaning

Removed duplicate records

Standardized categorical values (e.g., M → Male, S → Single)

Ensured consistent formatting

2. Data Transformation

Created Age Brackets using IF statements

Improved readability of categorical variables

Prepared data for analysis

3. Data Analysis

Built Pivot Tables to analyze:

Income vs Purchase Behavior

Age Group vs Sales

Region-based trends

4. Dashboard Creation

Designed an interactive dashboard with:

Filters (Slicers)

Charts

Key insights

📈 Key Insights

Certain age groups show higher purchase rates

Income level influences buying behavior

Regional differences impact sales trends
